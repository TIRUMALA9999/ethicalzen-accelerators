Icon files will be created automatically when you first load the extension.

For now, we'll use the SVG files as placeholders.

To create proper PNG icons later, you can:
1. Use any online SVG to PNG converter
2. Or use this command if you have ImageMagick:
   convert icon16.svg icon16.png
   convert icon48.svg icon48.png
   convert icon128.svg icon128.png

The extension will still work with the SVG files in most cases.
